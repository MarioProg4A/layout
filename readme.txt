inicio
Cuando se carga la página, se muestra un formulario de inicio de sesión que solicita un nombre de usuario y una contraseña.
Cuando se envía el formulario, se verifica si el nombre de usuario es 'mario' y la contraseña es 'd1234'.
Si las credenciales son correctas, la página se actualiza con información temática sobre el juego Halo.
Se cambia el contenido de la página para mostrar información sobre la historia de Halo, el Spartan Master Chief, el Covenant y el Flood.
Se cambia el fondo de la página a imágenes relacionadas con Halo.
Se proporciona un botón para permitir al usuario volver a la página de inicio de sesión.
Si las credenciales no son correctas, se muestra un mensaje de error indicando que el nombre de usuario o la contraseña son incorrectos.
